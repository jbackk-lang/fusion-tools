# fusion-tools

Narzędzia do analizy danych z fuzji jądrowej (W7-X, JET, DIII-D) 
z wykorzystaniem modeli GIA, TIMDR, Λ–τ–ρ oraz Modelu J.

## Moduły

- parsers/ — wczytywanie danych (HDF5, MDSplus, CSV)
- timdr/ — filtr redukcji informacji i wykrywania defektów
- latro/ — analiza strukturalna Λ–τ–ρ
- model_j/ — detekcja punktów skrętu sygnału
- demo/ — przykładowy notebook z analizą danych plazmowych

## Cel

Ułatwić analizę diagnostyk plazmy:
- IR camera
- Mirnov coils
- bolometry
- sygnały ECRH/NBI
- turbulencje i rezonanse

Projekt jest otwarty i rozwijany modułowo.
